# communication_module
## Get started
### Install dependencies
```bash
pip install openai==0.28
pip install gradio
```
### Run the code
```bash
python Home.py
```
Then you can play with the interface on the browser by clicking the link in the terminal.
```
...
Running on local URL:  http://127.0.0.1:7860
...
```
